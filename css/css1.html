# Announcement [通告] 文档仅供参考 下方有详情链接

📒Hi,all通告组件需求，来自于之前的一个业务场景，当时的需求是机房要做熔灾演练，需要提前发公告在网站上，此时接到需求的我秉持着不造轮子的原则，在网上大量找，也有，但是发现很多都是用JQ写的，而且扩展性也比较差，封装的相对不够完善，基于此，我便诞生了希望用react来做一个好玩的通告组件。
